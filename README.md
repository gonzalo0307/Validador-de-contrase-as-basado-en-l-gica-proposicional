Validador logico de contraseñas

Proyecto de Matematica Discreta que aplica la logica proposicional a la validación de contraseñas. El programa permite definir reglas de seguridad como presencia de letras, numeros, mayusculas, minusculas, simbolos, longitud minima o maxima, y condiciones de texto obligatorio o prohibido. Luego evalúa si una contraseña cumple la política definida y muestra un arbol logico con el resultado de cada condicion.

Tecnologias utilizadas

Python 3
Biblioteca estandar de Python (dataclasses, abc, typing, sys, os)

Interfaz por consola (CLI)

Programación orientada a objetos

Lógica proposicional aplicada a ciberseguridad básica

Funcionalidades

Validacion de contraseñas segun reglas logicas

Reglas para letras, numeros, mayusculas, minusculas y simbolos

Restricciones de longitud minima y maxima

Inclusion o exclusion de texto especifico

Visualizacion del resultado mediante arbol de evaluación lógica

Constructor rapido de reglas desde consola


Descripcion del proyecto

Este proyecto busca demostrar cómo los conceptos de logica proposicional y matematica discreta pueden aplicarse en un problema real de ciberseguridad: la validacion de contraseñas. De esta manera, se relaciona la teoría vista en clase con una implementacion practica en Python.
